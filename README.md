# TravelPlan

TravelPlan is a responsive web-based trip planner built strictly with frontend technologies as a learning project. It allows users to efficiently organize their travels by managing trips, tasks, expenses, and reservations from a single interface using client-side logic.

## Live Site
[View TravelPlan Online](#https://calebjt7.github.io/travel-organization/)

## Features
* Add and manage multiple trips with budget and description.
* Create and categorize to-do lists.
* Track expenses with categories and total calculation.
* Register bookings (flights, hotels, tours, etc.).
* Toggle between light and dark mode.
* Fully responsive design.
* Toast-style notifications for user feedback.

## Upcoming Improvements
* User authentication system (register/login).
* Save user data persistently via backend integration.
* Multi-user support with personal dashboards.

## Technologies Used
* HTML5
* CSS3 (custom properties, animations, responsive layout)
* JavaScript (DOM manipulation, modular architecture)
* Font Awesome icons & Google Fonts (Inter)
